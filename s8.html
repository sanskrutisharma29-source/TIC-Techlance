AI Doctor Clinic

📌 Project Summary

This project showcases the final UI slide of an AI-based healthcare system designed for rural areas. It highlights key features, impact, and scalability of the solution.

🚀 Key Highlights

  • Targets 900M+ population with limited healthcare access
  • Works offline for remote areas
  • Supports early diagnosis
  • Low-cost and easy to use
  • Helps local health workers

🎯 Purpose

To improve healthcare accessibility in rural communities using AI technology.

🛠️ Tech Used

  • HTML
  • CSS
